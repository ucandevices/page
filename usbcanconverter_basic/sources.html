Uploading new software

UCCB have got embedded DFU USB bootloader. Currently only way to upload new firmware is to use DfuSe utility program supplied by ST microelectronics. To upload new firmware follows those steps:

  • Download and Install DfuSe package from ST
  • Download .dfu firmware file from this page
  • Connect UCCB and type in terminal b device should reset and new device should connect to Your pc
  • Install driver for DFU device, drivers are in \DfuSe v3.0.5\Bin\Driver folder
  • Run DfuSe you should see windows like this.
  • Click Choose... and select DFU firmware. Click, Upgrade and Leave DFU mode buttons.
  • DONE!
    • Additional info on DFU can be found in ST UM0412 User manual.
