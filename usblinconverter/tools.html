Tools

You can use ULC on any operating system with serial terminal program like for example putty. Description of all available commands can be found in About section. All commands are the same as for USB-CAN converter so all tools can be reused, see UCAN USB-CAN converter project for libs and tools. ULC is also compatible with LINUX CAN-UTILS and SOCKET CAN. Device have its own terminal written in JAVA, uCCBViewer.

UCCBViewer is GUI program for both uCAN CAN and LIN converters. For USB-LIN converter uCCBViewer functionalities are: frame reception, transmission, master table setup. To run this program Java Runtime Enviroment(JRE) needs to be installed. To run UCCBViewer on Linux remember to run this program as superuser. sudo java -jar uCCBViewer-2.1.jar
